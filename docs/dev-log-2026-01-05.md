# 開発ログ 2026-01-05

## 概要

ポートフォリオサイトの初期構築を行った。

## 実施内容

### 1. Tailwind CSS の導入

- `@tailwindcss/vite` をインストール
- `vite.config.ts` にプラグイン追加
- `src/index.css` を Tailwind 用に更新

### 2. トップページ (`/`) の作成

以下のセクションを持つシンプルなポートフォリオページを作成:

- **Header**: サイト名 + SNSリンク（GitHub, Twitter, Zenn, LinkedIn）
- **Profile**: 丸いアイコン（`assets/github-icon.png`）+ 名前 + 肩書き
- **About**: 自己紹介セクション
- **Career**: 経歴セクション
- **Articles**: 寄稿記事セクション
- **Footer**: コピーライト

### 3. コンポーネント分割

`src/components/` に以下のコンポーネントを作成:

- `Header.tsx` - ヘッダー（サイト名 + SNSリンク）
- `Footer.tsx` - フッター（コピーライト）
- `ProfileSection.tsx` - プロフィール（アイコン + 名前）
- `AboutSection.tsx` - 自己紹介
- `CareerSection.tsx` - 経歴
- `ArticlesSection.tsx` - 寄稿記事

### 4. デザイン方針

- シンプル・ミニマル
- モバイルファースト（`sm:` ブレークポイントでレスポンシブ対応）
- 白背景 + グレーのアクセント
- 全体的に大きめの文字サイズ

### 5. コンテンツ追加

- SNSリンクに実際のURLを設定（GitHub, Twitter, Zenn, LinkedIn）
- About セクションの内容を記入
- Career セクションの内容を記入（神奈川大学）
- Articles セクションに Excite Tech Blog への寄稿記事を追加

## 技術スタック

- React 19 + TypeScript
- Vite 7 (SWC)
- Tailwind CSS 4
- Cloudflare Pages
